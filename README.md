# ShoppingmallProject
통합 쇼핑몰 솔루션 프로젝트

## 📖목차 
- 프로젝트 개요
- 개발환경
- 구현기능 / 상세설명

### 프로젝트 개요
다양한 오픈마켓 API를 사용하여 쇼핑몰을 통합해 손쉬운 쇼핑몰 활용이 가능하다.
